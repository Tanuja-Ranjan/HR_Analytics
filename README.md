# 👨‍💼 HR Analytics – Employee Attrition Analysis

## 📌 Overview

This project analyzes employee attrition using **Python and Power BI** to identify workforce trends and factors influencing employee turnover. It follows an end-to-end analytics workflow, including data cleaning, exploratory data analysis, and interactive dashboard development to support data-driven HR decisions.

---

## 🎯 Business Problem

Employee attrition is one of the biggest challenges for organizations as it impacts productivity, hiring costs, and employee retention.

This project aims to answer questions such as:

- Which departments have the highest attrition?
- Does overtime influence employee turnover?
- Which job roles and age groups experience higher attrition?
- How does salary relate to employee retention?
- What insights can help HR improve workforce retention?

---

## 🛠️ Tech Stack

- Python (Pandas, NumPy)
- Power BI
- Jupyter Notebook

---

## 📂 Project Workflow

### 1. Data Preparation & EDA (Python)

- Imported and explored the HR dataset.
- Cleaned missing and inconsistent values.
- Performed Exploratory Data Analysis (EDA).
- Prepared the dataset for visualization.

### 2. Dashboard Development (Power BI)

Built an interactive HR dashboard to analyze:

- Employee Attrition
- Department-wise Analysis
- Job Role Analysis
- Overtime Analysis
- Salary Analysis
- Employee Demographics
- Interactive Filters & Slicers

---

## 📊 Key Insights

- Identified departments and job roles with the highest attrition.
- Analyzed the relationship between overtime and employee turnover.
- Compared attrition across salary levels and employee demographics.
- Highlighted workforce trends to support employee retention strategies.
